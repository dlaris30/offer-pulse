## First Run Errors and Learnings

### Error 1 — PromoCode casing
- Generated: `disc15348` (lowercase, from implementer comment)
- Correct: `DISC15348` (all caps, from ticket body)
- **Rule:** When ticket body and implementer comment differ on casing, the ticket body is authoritative.

### Error 2 — StartDate interpretation
- Generated: `2026-03-16` (campaign start date — "From March 16")
- Correct: `2026-03-03` (deployment date — when the ticket was created and code activated)
- **Rule:** StartDate = the date the promo is deployed/activated, NOT the date the campaign goes live to customers. The cart team controls when customers see it; the promo just needs to exist in the system. When a ticket says "From [date]", that is the cart team's implementation date, not the promo StartDate.

### Error 3 — Name/Description wording
- Generated: `.uk 1yr registration cart cross-sell at 1.99 GBP PRICING-15348`
- Correct: `.UK cart cross-sell promo 1.99 GBP`
- **Rule:** Name/Description should be concise and match what the implementer would write — product/surface/price. Do not include the ticket number unless the implementer's style for that product line includes it (e.g., DISCWAMBA does; most don't). Match the capitalization of the product name as written (.UK not .uk).
